# 八字万年历

## 计算原则

### 八字计算

#### 年柱

* 年柱以 **立春** 为岁首。
* 出生于立春之前者，依前一年的干支计算。
* 出生于立春之后者，依当年的干支计算。
* 立春时刻采用韩国天文研究院资料。
* 计算式：**(公历年 - 4) % 60** = **六十甲子**索引

#### 月柱

* 月柱以 **二十四节气**中的 **十二节** 为准。
* 节入时刻精确套用。
* 使用节气：**立春（寅）、惊蛰（卯）、清明（辰）、立夏（巳）、芒种（午）、小暑（未）、立秋（申）、白露（酉）、寒露（戌）、立冬（亥）、大雪（子）、小寒（丑）**
* 节入之前，沿用前一月干支。
* 节入之后，改用当月干支。

#### 日柱

* 基准日：**1900年1月1日 = 甲戌日**
* 以基准日起算之经过日数对 60 取余计算。
* 日期更替以 **00:00** 为准。

#### 时柱

* 地支时辰对应：

  * **23:00-01:00** = **子时**
  * **01:00-03:00** = **丑时**
  * **03:00-05:00** = **寅时**
  * **05:00-07:00** = **卯时**
  * **07:00-09:00** = **辰时**
  * **09:00-11:00** = **巳时**
  * **11:00-13:00** = **午时**
  * **13:00-15:00** = **未时**
  * **15:00-17:00** = **申时**
  * **17:00-19:00** = **酉时**
  * **19:00-21:00** = **戌时**
  * **21:00-23:00** = **亥时**
* 子时分法：

  * **23:00-24:00** = **夜子时** — 当日
  * **00:00-01:00** = **早子时** — 当日
* 时干依日干与时支推算。

#### 时间校正

* 韩国夏令时间：

  * 1948-1960：+1小时
  * 1987-1988：+1小时
* 不采用真太阳时与均时差，使用法定时间。

### 大运计算

#### 起运岁数

1. 判定 **顺行** 或 **逆行**

   * **顺行**：阳干男、阴干女
   * **逆行**：阳干女、阴干男

2. 计算距离

   * **顺行**：由出生时刻起算至下一个节气
   * **逆行**：由出生时刻起算至上一个节气

3. 换算规则

   * 3日 = 1年
   * 1日 = 4个月
   * 1小时 = 5日
   * 起运岁数 = 日数 ÷ 3

#### 大运干支

* **顺行**：自月柱起依序推进
* **逆行**：自月柱起逆序推进
* 每十年一换

### 岁运

* 依 **六十甲子** 循环
* 显示范围：以当前年份为准，前10年至后10年
* 年龄以实岁计算
* 当前年份高亮显示

### 神煞与空亡

#### 主要贵人与吉神

* **天乙贵人**：以日干为准
  **甲戊庚-丑未 / 乙己-子申 / 丙丁-亥酉 / 辛-寅午 / 壬癸-卯巳**
* **文昌贵人**：以各日干食神之禄地为准
  **甲-巳 乙-午 丙-申 丁-酉 戊-申 己-酉 庚-亥 辛-子 壬-寅 癸-卯**
* **天德 / 月德贵人**：以月支为准
  **春-丙/丁、夏-庚/辛、秋-壬/癸、冬-甲/乙**

#### 三合系神煞

* **驿马煞**：三合首支所冲之地支
  **亥卯未-巳 / 寅午戌-申 / 巳酉丑-亥 / 申子辰-寅**
* **桃花煞**：三合之沐浴位
  **亥卯未-子 / 寅午戌-卯 / 巳酉丑-午 / 申子辰-酉**
* **华盖煞**：三合之库位
  **亥卯未-未 / 寅午戌-戌 / 巳酉丑-丑 / 申子辰-辰**

#### 主要煞

* **空亡**：日柱所在旬中所缺之地支
  **甲子旬-戌亥 / 甲戌旬-申酉 / 甲申旬-午未 / 甲午旬-辰巳 / 甲辰旬-寅卯 / 甲寅旬-子丑**
* **羊刃煞**：各日干之极旺之支
  **甲-卯 / 乙-寅 / 丙-午 / 丁-巳 / 戊-午 / 己-巳 / 庚-酉 / 辛-申 / 壬-子 / 癸-亥**
* **魁罡煞**：日柱为
  **庚辰、庚戌、壬辰、戊戌**
* **白虎大煞**：适用组合为
  **甲辰、乙未、丙戌、丁丑、戊辰、壬戌、癸丑**
* **元辰煞 / 鬼门关煞**：特定不和地支组合
  **子未、丑午、寅巳 / 寅未、卯辰 / 卯申、申亥、酉戌 / 酉午 等**

### 十神

* 以日主为基准，依五行生克关系判定

  * **比劫**：同我五行

    * **比肩** = 同阴阳
    * **劫财** = 异阴阳
  * **食伤**：我所生之五行

    * **食神** = 同阴阳
    * **伤官** = 异阴阳
  * **财星**：我所克之五行

    * **偏财** = 同阴阳
    * **正财** = 异阴阳
  * **官星**：克我之五行

    * **七杀** = 同阴阳
    * **正官** = 异阴阳
  * **印星**：生我之五行

    * **偏印** = 同阴阳
    * **正印** = 异阴阳
* 地支十神以主藏干为准。
* 藏干十神分别计算本气、中气、余气。

### 天干合冲

* **五合**
  **甲己→土，乙庚→金，丙辛→水，丁壬→木，戊癸→火**
* **冲**
  **甲庚冲、乙辛冲、丙壬冲、丁癸冲**
* **戊、己**不成冲

### 地支合冲

* **六合**
  **子丑→土，寅亥→木，卯戌→火，辰酉→金，巳申→水，午未→土**
* **三合**
  **申子辰→水局，亥卯未→木局，寅午戌→火局，巳酉丑→金局**
* **六冲**
  **子午冲、丑未冲、寅申冲、卯酉冲、辰戌冲、巳亥冲**
